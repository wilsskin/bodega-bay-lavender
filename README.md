# Bodega Bay Lavender
Shopify storefront for a family lavender ranch on the Sonoma Coast.

[bodegabaylavender.com](https://bodegabaylavender.com)

---

## What It Does
- Custom Shopify theme built for a small e-commerce business
- Product pages, collection browsing, and checkout optimized for seasonal sales
- Fully responsive across mobile and desktop

## How It's Built
Custom Shopify theme built in Liquid with CSS and JavaScript. Component architecture follows Shopify's sections and blocks model, allowing modular layout control through the theme editor.

**Stack:** Shopify, Liquid, CSS, JavaScript
